Как должен выглядеть файл настроек:

-----------------------------------
WIDTH=X
HEIGHT=X
FPS=X
R0=Y
G0=Y
B0=Y
R1=Y
G1=Y
B1=Y
MIN=X
MAX=X
BOT=X
TOP=X
WORLD_BORDERS=Z
-----------------------------------


где X - число, превосходящее ноль
    Y - число, превосходящее ноль, но не превосходящее 255
    Z - число, превосходящее ноль, но не превосходящее 1

WIDTH - ширина окна
HEIGHT - высота окна
FPS - кол-во кадров в секунду
COLOR1 - цвет "живых" клеток
COLOR2 - цвет "мёртвых" клеток
MIN - минимальное число соседей вокруг клетки, необходимое для зарождения в ней жизни
MAX - максимальное число соседей, при котором может зародиться жизнь
BOT - минимальное кол-во соседей, при котором "жизнь" продолжается в клетке
TOP - максимальное кол-во соседей, при котором "жизнь" продолжается в клетке
